Easily and instantly add Shopper Approved to your website with some simple short code from this plugin.

== Description ==

Shopper Approved for Wordpress allows you to instantly plugin your Shopper Approved seal and thank you page code right into your website powered by Wordpress.

You can also easily integrate the search engine schema code directly into your site by simply inserting your API token from Shopper Approved... No Programming Needed!

Not a Shopper Approved customer? Start collecting ratings and reviews now (http://www.shopperapproved.com) with the 30 day free trial.


== Installation ==

1. Upload `shopper-approved` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


1. Install Shopper Approved either via the WordPress.org plugin directory, or by uploading the `shopper-approved` folder to the `/wp-content/plugins/` directory on your server.
2. After activating Shopper Approved for Wordpress you will want to click on Shopper Approved on the left side menu.
3. Once there just start typing your Shopper Approved website in the box and then select your website from the menu when it appears.
4. Save the form and you will get the short code you can place in either your individual pages/posts or inside your site template. 

Note: when placing the short code inside of your template you will want to insert it like this: <?php echo do_shortcode('[shopper-approved-seal size="small"]'); ?>
